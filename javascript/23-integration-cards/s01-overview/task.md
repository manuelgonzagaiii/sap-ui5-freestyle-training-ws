# Integration Cards

> **Track:** JavaScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- sap.ui.integration cards: List/Table/Object/Analytical/Component
- Card manifest structure and data binding
- Host actions and card extensions
- Embedding cards in a freestyle app dashboard

## Planned stages
1. Add a List card bound to the orders OData service
2. Add an Object card for a selected customer
3. Add an Analytical card (chart)
4. Wire host actions (navigation) from a card
5. Assemble a card dashboard page

## SAPUI5 features covered
- `sap.ui.integration.widgets.Card`
- `card manifest`
- `host actions`
- `card extensions`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
