# Theming & Styling

> **Track:** JavaScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Theming parameters (Theme/Parameters.get) and Horizon theme family
- Custom CSS, library CSS and content density (Cozy/Compact)
- Dark mode and high-contrast themes
- Building a custom theme with UI5 Theming (overview)

## Planned stages
1. Read theme parameters at runtime and apply them
2. Add scoped custom CSS via a library/style resource
3. Toggle content density (Cozy/Compact) globally
4. Switch between sap_horizon / _dark / _hcb
5. Outline a custom theme build (theming overview)

## SAPUI5 features covered
- `Theming Parameters`
- `Horizon themes`
- `content density`
- `custom CSS`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
