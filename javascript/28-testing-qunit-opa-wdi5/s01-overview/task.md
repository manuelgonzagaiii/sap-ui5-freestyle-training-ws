# Testing: QUnit, OPA5 & wdi5

> **Track:** JavaScript  ·  **Phase F · Quality & Delivery**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- QUnit unit tests for formatters/controllers/models
- OPA5 integration tests with page objects and the test starter
- Recording tests with the Test Recorder
- End-to-end testing with wdi5 and the Karma runner

## Planned stages
1. Write QUnit tests for formatter + model logic
2. Set up the test starter (Test.qunit.html / testsuite)
3. Author an OPA5 journey with page objects
4. Generate steps with the Test Recorder
5. Add a wdi5 e2e smoke test

## SAPUI5 features covered
- `QUnit`
- `OPA5 + page objects`
- `Test Recorder`
- `wdi5 / Karma + coverage`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
