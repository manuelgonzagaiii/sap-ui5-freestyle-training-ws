# UI5 Web Components Interop

> **Track:** JavaScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- What @ui5/webcomponents are and how they differ from sap.m
- Embedding web components in a UI5 freestyle app
- Interop: properties, events and slots from XML views
- When to choose web components vs classic controls

## Planned stages
1. Add @ui5/webcomponents and load a component
2. Embed a web component in an XML view via custom namespace
3. Bind data and handle a web-component event
4. Compare with the equivalent sap.m control
5. Document interop guidance

## SAPUI5 features covered
- `@ui5/webcomponents`
- `custom elements interop`
- `events/slots`
- `selection guidance`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
