# Data Binding & Models

> **Track:** JavaScript  ·  **Phase B · UI Composition**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- JSONModel and the three binding types: property, aggregation, element
- Binding modes: OneWay, TwoWay, OneTime and default model mode
- List binding with templates, sorters and filters
- Relative vs absolute binding paths and binding contexts

## Planned stages
1. Load products into a JSONModel and bind a List with a template
2. Bind detail fields via element binding (bindElement)
3. Switch model to TwoWay and edit through Inputs
4. Apply a Sorter and a Filter to the list binding
5. Bind aggregations with factory functions for heterogeneous items

## SAPUI5 features covered
- `sap.ui.model.json.JSONModel`
- `property/aggregation/element binding`
- `binding modes`
- `Sorter/Filter`
