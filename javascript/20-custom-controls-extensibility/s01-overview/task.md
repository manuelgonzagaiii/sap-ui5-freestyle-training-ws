# Custom Controls & Extensibility

> **Track:** JavaScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Authoring a custom control: metadata, renderer, init
- Extending existing controls and overriding the renderer
- EventBus, component reuse and OverrideExecution
- Packaging controls in a library

## Planned stages
1. Build a custom control (e.g. a rating/indicator) with a renderer
2. Extend an existing control to add behaviour
3. Publish/subscribe across views with the EventBus
4. Reuse a sub-component (ComponentContainer reuse)
5. Use OverrideExecution to extend a controller hook

## SAPUI5 features covered
- `sap.ui.core.Control + Renderer`
- `control extension`
- `core.EventBus`
- `component reuse`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
