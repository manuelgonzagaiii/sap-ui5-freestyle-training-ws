# Dialogs, Popovers & Messaging

> **Track:** JavaScript  ·  **Phase C · Interaction**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Dialog, Popover, ActionSheet and ResponsivePopover
- Transient feedback: MessageToast, MessageBox, BusyDialog/BusyIndicator
- The Message Manager and MessagePopover for validation messages
- MessageStrip and inline messaging

## Planned stages
1. Open a confirmation Dialog from a fragment
2. Show a Popover anchored to a control
3. Use MessageToast + MessageBox for feedback
4. Register the core MessageManager and bind a MessagePopover
5. Surface a BusyIndicator during an async action

## SAPUI5 features covered
- `sap.m.Dialog/Popover/ActionSheet`
- `MessageToast/MessageBox`
- `core.message.MessageManager`
- `MessagePopover`
