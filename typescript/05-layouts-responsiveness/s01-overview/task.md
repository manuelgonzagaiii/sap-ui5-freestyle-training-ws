# Layouts & Responsiveness

> **Track:** TypeScript  ·  **Phase B · UI Composition**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Flex layouts: VBox, HBox, FlexBox and alignment
- sap.ui.layout: Grid, Splitter, ResponsiveSplitter, FixFlex
- CSS grid via sap.ui.layout.cssgrid and sap.f.GridContainer
- Responsive/adaptive design, breakpoints and form layouts

## Planned stages
1. Lay out content with VBox/HBox and alignItems/justifyContent
2. Build a responsive Grid with default span/indent
3. Add a ResponsiveSplitter master/detail shell
4. Use a CSS grid container for a tile dashboard
5. Make it adaptive with sap.ui.Device + breakpoint classes

## SAPUI5 features covered
- `sap.m.FlexBox/VBox/HBox`
- `sap.ui.layout.Grid/Splitter`
- `cssgrid / GridContainer`
- `sap.ui.Device`
