# Drag & Drop, Uploads, Wizards & Charts

> **Track:** TypeScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Drag & drop with DragInfo/DropInfo and rearrange
- File upload with UploadSet / FileUploader
- Multi-step flows with Wizard
- Visualisation: micro charts and sap.viz/VizFrame

## Planned stages
1. Enable drag & drop reordering in a list/table
2. Add an UploadSet for order attachments
3. Build a Wizard for order creation
4. Add micro charts to the object header
5. Render a VizFrame chart for analytics

## SAPUI5 features covered
- `dnd DragInfo/DropInfo`
- `UploadSet/FileUploader`
- `sap.m.Wizard`
- `micro charts / sap.viz`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
