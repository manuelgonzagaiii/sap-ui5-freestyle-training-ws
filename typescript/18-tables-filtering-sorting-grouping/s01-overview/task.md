# Tables, Filtering, Sorting & Grouping

> **Track:** TypeScript  ·  **Phase D · Data & Backend**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Choosing between sap.m.Table, sap.ui.table.Table, TreeTable, AnalyticalTable
- Growing/threshold, sticky headers and cell types
- ViewSettingsDialog for sort/group/filter and a basic FilterBar
- Export to spreadsheet (sap.ui.export)

## Planned stages
1. Compare responsive Table vs grid Table for the orders list
2. Add growing + sticky column headers
3. Wire a ViewSettingsDialog for sort/group/filter
4. Add a simple search + filter toolbar
5. Export the table via sap.ui.export.Spreadsheet

## SAPUI5 features covered
- `sap.m.Table / sap.ui.table.*`
- `growing / sticky`
- `ViewSettingsDialog`
- `sap.ui.export`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
