# TypeScript Setup & Tooling

> **Track:** TypeScript  ·  **Phase A · Foundations**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Why TypeScript for UI5 and the official type packages (@sapui5/types)
- tsconfig.json, ui5-tooling-transpile and the babel/ts toolchain
- Type-safe controllers, views and models (the 'extend' + override pattern)
- ESLint for TypeScript and the @ui5/ts-interface-generator for custom controls

## Planned stages
1. Add tsconfig.json + @sapui5/types and configure paths
2. Wire ui5-tooling-transpile middleware/task in ui5.yaml
3. Convert Component + a controller to a typed class
4. Set up ESLint (typescript-eslint) and type checking in CI
5. Generate control interfaces with @ui5/ts-interface-generator

## SAPUI5 features covered
- `@sapui5/types`
- `ui5-tooling-transpile`
- `typed controllers/views`
- `ts-interface-generator`
