# Flexibility, Adaptation & Variants

> **Track:** TypeScript  ·  **Phase E · Advanced & Rare**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- UI5 flexibility services and key-user Runtime Adaptation (RTA)
- VariantManagement and SmartVariantManagement
- Personalization (p13n) and persistence
- Adaptation vs personalization layers

## Planned stages
1. Enable flexibility + RTA key-user adaptation
2. Add VariantManagement to a filter bar / table
3. Persist personalization with the p13n engine
4. Save and apply a variant
5. Reset and manage variants

## SAPUI5 features covered
- `sap.ui.fl (flexibility)`
- `RTA key-user adaptation`
- `VariantManagement`
- `p13n engine`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
