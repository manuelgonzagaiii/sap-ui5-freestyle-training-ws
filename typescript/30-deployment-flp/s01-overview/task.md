# Fiori Launchpad & Delivery

> **Track:** TypeScript  ·  **Phase F · Quality & Delivery**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- Fiori Launchpad sandbox: shell, tiles, target mappings
- Cross-app navigation and intents
- App descriptor for FLP (sap.app crossNavigation, sap.flp)
- Deployment options without paid systems (static hosting overview)

## Planned stages
1. Run the app inside the FLP sandbox (flpSandbox.html)
2. Define a tile + target mapping (semantic object/action)
3. Trigger cross-app navigation via intents
4. Finalise sap.app/sap.ui5/sap.flp descriptor metadata
5. Build + serve the self-contained app on static hosting

## SAPUI5 features covered
- `FLP sandbox`
- `tiles / target mappings`
- `cross-app navigation (intents)`
- `static hosting`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
