# Internationalisation & Formatting

> **Track:** TypeScript  ·  **Phase B · UI Composition**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- ResourceModel + i18n.properties and the i18n>key syntax
- Locale-aware Number/Date/Currency/Unit formatters
- Right-to-left (RTL) and multiple language bundles
- Pluralisation, parameters and message bundle conventions

## Planned stages
1. Register the i18n ResourceModel and externalise all texts
2. Format dates and numbers with sap.ui.core.format.* formatters
3. Add currency + unit formatting with the data binding type
4. Provide a second language bundle and switch locale
5. Verify RTL rendering with sap-ui-rtl

## SAPUI5 features covered
- `sap.ui.model.resource.ResourceModel`
- `DateFormat/NumberFormat`
- `Currency/Unit types`
- `RTL`
