# Value Help & Smart Controls

> **Track:** TypeScript  ·  **Phase D · Data & Backend**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- ValueHelpDialog and field value help patterns
- Annotation-driven sap.ui.comp controls in freestyle: SmartField, SmartTable, SmartFilterBar
- Value lists from OData annotations
- When to reach for smart controls vs plain controls

## Planned stages
1. Add a customer value help with a ValueHelpDialog
2. Drive a SmartField from OData annotations
3. Render a SmartTable + SmartFilterBar over the orders entity set
4. Provide value lists via annotations (or local annotation file)
5. Mix smart + freestyle controls in one view

## SAPUI5 features covered
- `sap.ui.comp.valuehelpdialog`
- `SmartField/SmartTable/SmartFilterBar`
- `ValueList annotations`

## Backend for this lesson
**SAP CAP + SQLite (OData V4) + annotations**
