# Views & Controllers

> **Track:** TypeScript  ·  **Phase A · Foundations**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- XML views as the recommended view type, plus JS/JSON/HTML/Typed views
- Controllers: lifecycle hooks (onInit, onBeforeRendering, onAfterRendering, onExit)
- Accessing controls with byId, the view's model, and event handlers
- View/controller naming conventions and separation of concerns

## Planned stages
1. Create App.view.xml + App.controller.js wired as rootView
2. Add a Main view and move content out of the component root
3. Implement lifecycle hooks and log the sequence
4. Wire a button press handler and update a JSONModel property
5. Compare view types: add a tiny JS view and a Typed view variant

## SAPUI5 features covered
- `XML/JS/JSON/HTML/Typed views`
- `sap.ui.core.mvc.Controller`
- `lifecycle hooks`
- `byId / event handlers`
