# UI5 Tooling & Build

> **Track:** TypeScript  ·  **Phase F · Quality & Delivery**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- ui5 serve vs ui5 build and the build pipeline
- Custom middleware and custom tasks
- Transpilation (ui5-tooling-transpile) and linting (eslint, ui5-linter)
- Self-contained build, cache buster and Component-preload

## Planned stages
1. Configure ui5.yaml middleware (proxy to CAP, livereload)
2. Add a custom task to the build
3. Wire eslint + ui5-linter into the project
4. Produce a production build (ui5 build --all)
5. Verify cache buster + preload bundles

## SAPUI5 features covered
- `UI5 Tooling (serve/build)`
- `custom middleware/tasks`
- `ui5-linter / eslint`
- `self-contained build`

## Backend for this lesson
**SAP CAP + SQLite (OData V4)**
