# Fragments & UI Reuse

> **Track:** TypeScript  ·  **Phase C · Interaction**
> **Status:** Skeleton — stages to be populated.

This is a **guided-project (framework) lesson**: your app code carries forward
from one stage to the next, and from the previous lesson into this one. This
overview page is the roadmap for the lesson; each numbered item below will
become a hands-on stage.

## What you'll learn
- XML/JS fragments and the Fragment factory
- Reusing UI across views and lazy-instantiating fragments
- Fragment controllers and id scoping (createId)
- BlockLayout and reusable content blocks

## Planned stages
1. Extract a reusable form into an XML fragment
2. Instantiate a fragment lazily and cache it on the controller
3. Scope fragment ids correctly with this.getView().byId
4. Reuse the fragment across two views
5. Compose a BlockLayout from blocks

## SAPUI5 features covered
- `sap.ui.core.Fragment`
- `XML/JS fragments`
- `fragment id scoping`
- `sap.uxap/BlockLayout`
